Git Comands:

# Creating the text file with the important Git commands
git_commands = """
1. Setting Up Git (One-time Setup)
   - Set your name:
     git config --global user.name "Your Name"
   - Set your email:
     git config --global user.email "youremail@example.com"

2. CREATING and Cloning Repositories
   - Initialize a new Git repository:
     git init
   - Clone an existing repository:
     git clone <repository_url>

->>>>>
git init
git add .
git commit -m "Commit inicial"

git remote add origin https://github.com/teu-username/nome-do-repo.git

git branch -M main
git push -u origin main


Caso haja algum conflito inicial (ReadMe)

git push --force origin main

<<<<<-
3. Checking Repository Status
   - Check the status of your files (modified, staged, etc.):
     git status
   - Show detailed information about a file:
     git diff <file_name>

4. Staging and Committing Changes
   - Stage a file for commit:
     git add <file_name>
   - Stage all changes:
     git add .
   - Commit your staged changes:
     git commit -m "Commit message describing changes"

5. Viewing Commit History
   - Show the commit history of the current branch:
     git log
   - Show a simplified, one-line history:
     git log --oneline

6. Working with Branches
   - Create a new branch:
     git branch <branch_name>
   - List all branches:
     git branch
   - Switch to another branch:
     git checkout <branch_name>
   - Create a new branch and switch to it:
     git checkout -b <branch_name>
   - Merge another branch into the current branch:
     git merge <branch_name>

7. Remote Repositories
   - Add a remote repository:
     git remote add origin <repository_url>
   - Push changes to a remote repository:
     git push origin <branch_name>
   - Pull the latest changes from the remote repository:
     git pull origin <branch_name>

8. Undoing Changes
   - Unstage a file (after git add):
     git reset <file_name>
   - Undo changes in a file (before committing):
     git checkout -- <file_name>
   - Undo the last commit (keep changes in the working directory):
     git reset --soft HEAD~1
   - Completely remove the last commit (including changes):
     git reset --hard HEAD~1

9. Deleting Files
   - Remove a file from Git (but keep it locally):
     git rm --cached <file_name>
   - Remove a file from both Git and the local system:
     git rm <file_name>

10. Tagging
   - Create a tag:
     git tag <tag_name>
   - Push tags to remote:
     git push origin <tag_name>

11. Stashing Changes
   - Stash uncommitted changes (save them temporarily):
     git stash
   - List all stashes:
     git stash list
   - Apply the most recent stash:
     git stash apply
   - Remove a stash:
     git stash drop

12. Rebasing (Advanced)
   - Rebase the current branch onto another branch:
     git rebase <branch_name>
"""

# Save the Git commands to a text file
file_path = "/mnt/data/important_git_commands.txt"
with open(file_path, "w") as file:
    file.write(git_commands)
